# Hadmin

概述：.NET 通用后台管理系统，前后的分离项目，后端代码采用模块化开发



## 系统架构

### 后端

- dotnet8
- webapi
- sqlsguar
- 数据库 postgresql
- 

### 前端

- pnpm
- vue3
- pinia
- vue-router
- ant-design-vue ui


## 快速启动

### 前端

1. npm install -g pnpm  (安装pnpm)
2. pnpm init -y (初始化pnpm，有package-lock.json文件则不需要初始化)
3. pnpm dev (直接启动)

### 后端

1. 数据库初始化创建对应数据库名称，执行sql脚本在doc文件夹下（先导入init-postgresql.sql，再导入后面sql脚本）
2. 安装dotnet8环境
3. 直接启动后端 dotnet run
